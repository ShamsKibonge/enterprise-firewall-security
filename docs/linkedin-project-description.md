# LinkedIn Project Description

Enterprise Firewall Security Implementation with Palo Alto NGFW

I built a professional cybersecurity portfolio case study documenting an enterprise Palo Alto Networks NGFW implementation in a controlled lab environment. The project covers firewall configuration management, administrator authentication, security zones, least privilege security policy, NAT, App-ID, threat prevention, URL filtering, WildFire analysis, User-ID, SSL decryption, and monitoring/reporting.

The repository includes architecture diagrams, implementation summaries, validation evidence mapping, lessons learned, resume bullets, and interview preparation. The focus is on showing practical firewall/security engineering skills using sanitized evidence and honest documentation boundaries.

Key skills: Palo Alto NGFW, PAN-OS, security zones, NAT, App-ID, URL Filtering, WildFire, User-ID, SSL decryption, threat logs, reporting, and security validation.
