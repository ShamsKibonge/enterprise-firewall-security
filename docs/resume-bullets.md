# Resume Bullets

- Designed and documented an enterprise Palo Alto NGFW security implementation covering security zones, security policy, NAT, App-ID, threat prevention, URL filtering, WildFire, User-ID, SSL decryption, and reporting.
- Built a zone-based firewall architecture with routed segmentation concepts, controlled management access, and validated traffic flows using available lab evidence.
- Documented least privilege Security Policy and NAT workflows, including source NAT, destination NAT, policy hit-count review, and traffic-log validation.
- Applied App-ID design principles to control traffic by application instead of relying only on TCP/UDP ports.
- Documented Security Profile Group usage for Antivirus, Anti-Spyware, and Vulnerability Protection inspection on allowed traffic.
- Validated web-control objectives through URL Filtering evidence and documented blocked-category enforcement.
- Documented WildFire analysis workflow for unknown-file inspection and malware verdict review.
- Mapped User-ID and group-based access requirements to acquisition-user and marketing-user access scenarios.
- Documented SSL Forward Proxy decryption workflow, including certificate handling and no-decrypt exceptions for sensitive categories.
- Organized firewall validation evidence across Traffic, Threat, URL, WildFire, System, Configuration, ACC, Dashboard, and reporting workflows.
